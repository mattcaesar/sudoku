# sudoku
 A terminal-based Sudoku game in Ruby.

How to play:

1) Open game.rb
2) On line 70, edit 'FILE_NAME' to one of the following.
    sudoku1.txt
    
    sudoku2.txt
    
    sudoku3.txt
    
    sudoku1_almost.txt (for a nearly complete board)
    
    sudoku1_solved.txt (for a completed game )
    
3) Run game.rb in terminal
